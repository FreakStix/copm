# КиберМосква — лендинг проекта

Одностраничный сайт для регистрации участников и прохождения тестирования
по цифровой грамотности. Чистый HTML/CSS/JS, без сборщиков — можно сразу
публиковать на GitHub Pages, Netlify или Timeweb.

## 📅 Как заполнить расписание (афишу)

В `index.html`, секция `id="poster"`, найдите блоки `schedule-row`.
В каждом замените:
- `<span class="sched-day">—</span>` → число (например, `12`)
- `<span class="sched-month">—</span>` → месяц (например, `МАЯ`)
- `00:00` → время начала
- `Название мероприятия №1` → реальное название
- `Спикер уточняется · площадка уточняется` → ФИО спикера и место
